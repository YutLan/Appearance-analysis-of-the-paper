README

The source code of this system is opened in github Appearance-analysis-of-the-paper.

1. You can upload a paper (a pdf file) to the website, and the system will evaluate it from several dimensions.

2. The score is for reference only, and we hope it will be helpful.

3. The system will not save your PDFs.

Reference: Huang J B. Deep Paper Gestalt[J]. arXiv preprint arXiv:1812.08775, 2018.
